## 📘 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma página web em HTML e CSS dedicada à vida e às contribuições de John von Neumann, uma das mentes mais brilhantes do século XX.

A aplicação apresenta uma estrutura moderna e responsiva, organizada em seções que abordam desde sua infância até seu legado científico. O conteúdo inclui uma linha do tempo, descrição de sua carreira e explicações sobre suas principais contribuições em áreas como computação, matemática, física quântica e teoria dos jogos.

Além do aspecto informativo, o projeto também demonstra conceitos fundamentais de desenvolvimento web, como:

* Estruturação semântica com HTML
* Estilização com CSS
* Layout responsivo
* Organização de conteúdo em seções e navegação

O objetivo principal é unir aprendizado técnico com conteúdo educativo, criando uma experiência visual agradável e informativa sobre a história de John von Neumann.
